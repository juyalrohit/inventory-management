# API Docs

Base URL: `/api`

## Auth
- `POST /auth/register`
- `POST /auth/login`
- `GET /auth/me`

## Products
- `GET /products`
- `GET /products/:id`
- `POST /products`
- `PUT /products/:id`
- `DELETE /products/:id`

## Stock
- `GET /stock/movements`
- `POST /stock/adjust`

## Orders
- `GET /orders`
- `GET /orders/:id`
- `POST /orders`
- `PATCH /orders/:id/status`

## Dashboard
- `GET /dashboard/summary`
